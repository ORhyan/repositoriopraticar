Podemos também colocar listas nao ordenadas dentro de listas ordenadas, ficando assim:

  1. Material escolar
    • Caneta
    • Lapis
    • Caderno
  2. Alimentos
    • Frango
    • Tomate
    • Macarrão

Podemos também modificar o comportamento das listas,
basta usar o parametro"type", mudando e as deixando assim:


  1. type "a", para deixa-las e ordem alfabetica.
  2. Letra b
  3. Letra c

    4. Enfim, há 5 variações de type: 1, A, a, I, i

Para as listas não ordenadas, também, basta inserir o parametro "type"

  • type = "circle", faz as listas ficarem com o pontinho redondo
  • para deixar assim QUADRO, basta colocar o tipo = "square"
